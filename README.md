
# AI-Powered Resume Analyzer & Job Recommender

**Python + Tkinter project** to analyze resumes (PDF/DOCX), extract skills, match them with job descriptions, and recommend suitable roles.  

---

## Features

- Extracts skills from resumes automatically.  
- Matches resume keywords with job description.  
- Shows match percentage and missing keywords.  
- Suggests actionable improvements for resume.  
- Recommends top roles using AI-based similarity (TF-IDF + cosine similarity).  
- Supports PDF and DOCX resume files.  
- GUI built using Tkinter for easy use.
